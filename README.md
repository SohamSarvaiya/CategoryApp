This app demonstrates a clean and modern architecture for Android development as of December 2024. Built with Jetpack Compose, it features:

MVVM Architecture: Streamlined UI state management with ViewModels.
Dependency Injection: Powered by Hilt for a scalable and maintainable codebase.
StateFlow Integration: For reactive state updates and seamless UI rendering.
API Handling: Retrofit-backed repository pattern for efficient data retrieval.
Dynamic Navigation: Simple yet robust navigation that allows users to click on a category and explore its related items effortlessly.
